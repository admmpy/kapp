## API Endpoints Reference

### GET /api/health
Check backend status
```bash
curl http://localhost:5001/api/health
```

### GET /api/stats
Get learning statistics
```bash
curl http://localhost:5001/api/stats
```

### GET /api/cards/due
Get cards due for review
```bash
# All due cards
curl http://localhost:5001/api/cards/due

# Filtered by deck
curl http://localhost:5001/api/cards/due?deck_id=1

# With limit
curl http://localhost:5001/api/cards/due?limit=10
```

### POST /api/reviews
Submit a review
```bash
curl -X POST http://localhost:5001/api/reviews \
  -H "Content-Type: application/json" \
  -d '{
    "card_id": 1,
    "quality_rating": 4,
    "time_spent": 12.5
  }'
```

### GET /api/audio/{filename}
Get TTS audio file
```bash
curl http://localhost:5001/api/audio/abc123.mp3
```
